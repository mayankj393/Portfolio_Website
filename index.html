Resume Matcher

The Resume Matcher project is designed to automatically sort resumes based on their similarity to a given job description. The project utilizes a NLP Libraries like NLTK and Spacy to Preprocess the text, which provides robust natural language processing (NLP) capabilities. The key objectives of the project include calculating the similarity score between resumes and the job description using cosine similarity and extracting relevant information such as email addresses and phone numbers from the resumes.

Roles and Responsibilities:

  • Involved in Preprocessing of resumes and job description using NLTK libraries
  • Involved in Lemmatization, tokenization and removal of stop words from the resumes and job description.
  • Used TF-IDF vectorizer to convert words or terms into numerical vectors in order to represent textual data in a machine-readable format.
  • Deployed the application using the flask framework.
  • created a database in SQL-LITE that stores the data of resume matching for each user.

Question Answer Generator

The aim of the project is to generate the question and answers from the given context provided by the user. For generating the no repeating questions T5 base pretrained model is used which is very popular text to text generation model. For generating the answers from the questions Bert-base-uncase model is used which is transformer-based model. This app generates the question-and-answer pairs from the provided context paragraph. Stream-lit is used as a deployment platform for this project

Roles and Responsibilities:

  • involved in the training of both question-answer models.
  • Involved in the evaluation of the both models.
  • Created the application in the stream-lit platform to deploy the project.
